# Telegram reminder bot

Простой Telegram-бот на Python. Репозиторий содержит код бота, шаблон настроек:
`.env.example`
`settings.ini.example`

## Установка


## Настройка

Скопируйте файлы с настройками и заполните их

```bash
cp .env.example .env
nano .env
```

```bash
cp settings.ini.example settings.ini
nano settings.ini
```
